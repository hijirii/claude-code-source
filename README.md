# Claude Code Source

This is the extracted source code of [Claude Code](https://github.com/anthropics/claude-code) (v2.1.88), reconstructed from the source map file (`cli.js.map`) that was inadvertently published to npm.

**Disclaimer:** This code belongs to Anthropic. It was made public due to the source map leak and is provided here for educational and research purposes only.

## Source

The source was extracted using:
1. Download the npm package: `npm pack @anthropic-ai/claude-code`
2. Extract the `cli.js.map` file
3. Use the `source-map` library to reconstruct TypeScript source files
